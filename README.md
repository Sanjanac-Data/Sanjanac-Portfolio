# Sanjana Portfolio

A modern, responsive personal portfolio built with **HTML + Tailwind CSS + JavaScript**, deployed using **GitHub Pages**.

## Features
- Glassmorphism animated navbar with scroll blur
- Hero section with animated gradient border + stats + scroll indicator
- About section with info cards + tech badges
- Skills section with proficiency bars + experience badges
- Projects carousel with category filters + metrics
- Bento-style services grid with hover glow
- Testimonials carousel with grayscale-to-color photos + ratings
- Contact form with validation + alerts (Formspree-ready)
- Scroll reveal animations
- Responsive design (mobile-first)

## Tech Stack
- HTML
- Tailwind CSS (CDN)
- JavaScript
- Swiper.js (carousel)
- Lucide Icons

## Run Locally
Just open `index.html` in your browser.

(Recommended) Use VS Code + Live Server extension:
1. Install Live Server
2. Right-click `index.html` → **Open with Live Server**

## Deploy (GitHub Pages)
1. Go to **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: `main` / folder: `/ (root)`
4. Save and open the live link
